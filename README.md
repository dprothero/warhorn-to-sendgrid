## Environment variables required

Create a `.env` file with these variables:

```
WARHORN_TOKEN=[value of _destro_session cookie from logged in session]
WARHORN_EVENT_ID=[id from Warhorn urls for your event: warhorn.net/events/<this is the id>/...]
SENDGRID_TOKEN=[your SendGrid API Key]
SENDGRID_LIST_ID=[your SendGrid list id]
```
